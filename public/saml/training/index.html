Exercise Set 1: Single Dysfluency Types

These exercises contain single, clear dysfluency events. Focus on identifying the type and assigning severity.

Exercise 1.1: Repetitions

Audio transcript (duration shown in brackets):

"I want to [0.8s] b-b-b-baby. Can you [1.2s] p-please help me?"

Tasks:

  1. Identify both dysfluency events
  2. Determine the repetition type for each (ISR, SR, or Mixed)
  3. Assign severity ratings based on duration
  4. Write the shorthand annotation
Show Answer

Answer:

  • "b-b-b-baby": ISR (single sound repeated), 0.8s = severity 1
  • "p-please": ISR (single sound repeated), 1.2s = severity 2
I want to ( [R:isyl 1] b- b- b- ) baby. Can you ( [R:isyl 2] p- p- ) please help me?

Exercise 1.2: Prolongations

Audio transcript:

"I [1.5s] ssssaw the [0.4s] mmman."

Tasks:

  1. Identify both prolongation events
  2. Note the prolonged sound and position (initial/medial/final)
  3. Assign severity ratings
  4. Write the shorthand annotation
Show Answer

Answer:

  • "ssssaw": Prolongation on "s", initial position, 1.5s = severity 2
  • "mmman": Prolongation on "m", initial position, 0.4s = severity 0
I s( [P 2] s- )aw the m( [P 0] m- )an.

Exercise 1.3: Blocks

Audio transcript:

"Let's [2.1s silent pause] go to the [1.8s audible tension] store."

Tasks:

  1. Determine block type for each (IBC, ABC, IBV, ABV)
  2. Assign severity ratings
  3. Write the shorthand annotation
Show Answer

Answer:

  • Before "go": IBC (inaudible, consonant), 2.1s = severity 3
  • Before "store": ABC (audible, consonant), 1.8s = severity 2
Let's ( [IBC 3] ) go to the ( [ABC 2] ) store.

Exercise Set 2: Mixed Types and Edge Cases

These exercises contain multiple dysfluency types and require careful judgment.

Exercise 2.1: Multiple Dysfluency Types

Audio transcript:

"[1.0s silent pause] Um, I-I-I [0.6s] th-think that we should, you know, [2.5s] ggggo."

Tasks:

  1. Identify which events are stuttering dysfluencies (tag) vs. normal disfluencies (don't tag)
  2. Classify each stuttering event
  3. Write the complete annotation

Hints: "Um" and "you know" are filler words (don't tag). "I-I-I" is a monosyllabic word repetition. Consider whether the initial pause shows signs of struggle.

Show Answer

Answer:

  • Initial pause: If no audible struggle, likely a thinking pause - DON'T TAG. If struggle evident, tag as block.
  • "Um": Filler word - DON'T TAG (write as plain text)
  • "I-I-I": SR (monosyllabic word), 0.6s = severity 1
  • "th-think": ISR, 0.6s = severity 1
  • "you know": Filler phrase - DON'T TAG
  • "ggggo": Prolongation on "g", 2.5s = severity 3
Um, ( [R:syl 1] I- I- I- ) ( [R:isyl 1] th- th- ) think that we should, you know, g( [P 3] g- )o.

Exercise 2.2: ISR vs Mixed

Audio transcript:

"I want some [0.9s] t-t-to-to-toffee."

Question: Is this ISR or Mixed? Explain your reasoning.

Show Answer

Answer: MIXED

This is Mixed because:

  • "t-t" = incomplete syllable (ISR component)
  • "to-to" = complete syllable of "toffee" (SR component)
  • Both ISR AND SR are present → Mixed
I want some ( [R:mix 1] t- t- to- to- ) toffee.

Exercise 2.3: Intentional vs Involuntary

Classify each as Tag (stuttering) or Don't Tag (intentional/normal):

  1. 1. "No no no, that's not right" (speaker shaking head)
  2. 2. "I-I-I don't know" (speaker appears to struggle)
  3. 3. "The biiiig one" (emphasizing size)
  4. 4. "The [tense pause] big one" (speaker appears stuck)
  5. 5. "I went to... I mean I drove to the store"
Show Answer
  1. 1. Don't Tag - Intentional repetition for emphasis
  2. 2. Tag - SR (monosyllabic word repetition with struggle)
  3. 3. Don't Tag - Intentional prolongation for emphasis
  4. 4. Tag - Block (speaker appears stuck before word)
  5. 5. Don't Tag - Normal revision/correction (disfluency)
