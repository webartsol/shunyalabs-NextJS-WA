Integration Guides

Configure audio analysis tools for SAML annotation. Step-by-step setup for Audacity and Praat.

Audacity Configuration

Free, cross-platform audio editor

1 Installation

  1. Download from audacityteam.org
  2. Install for your platform (Windows, Mac, Linux)
  3. Launch Audacity

2 Recommended Settings

  1. Sample rate: Edit → Preferences → Quality → 44100 Hz
  2. Time display: Click time display → "hh:mm:ss + milliseconds"
  3. Spectrogram: Track dropdown → Spectrogram (optional)

Creating SAML Annotation Labels

  1. 1. Add label track: Tracks → Add New → Label Track
  2. 2. Create label: Select audio region, then Ctrl+B (Cmd+B on Mac)
  3. 3. Label naming: Use shorthand notation in labels
  4. 4. Export: File → Export → Export Labels

Label naming examples:

R:isyl 1
R:syl 2
P 3
IBC 4
ABC 2

Measuring Duration

  1. Zoom in: Ctrl+1 until individual sounds are visible
  2. Select: Click at start of dysfluency, drag to end
  3. Read: Duration appears in selection display (bottom of window)
  4. Precision: View → Zoom → Zoom to Selection

Praat Configuration

Specialized phonetics analysis tool

1 Installation

  1. Download from fon.hum.uva.nl/praat
  2. Open sound: Open → Read from file
  3. View: Select sound, click "View & Edit"

2 Creating TextGrid

  1. Select sound object
  2. Click "Annotate" → "To TextGrid"
  3. Create tiers for transcript, dysfluency, severity

TextGrid Tier Structure

Tier Type Name Content
1 Interval transcript Full text transcription
2 Interval dysfluency Dysfluency annotations (shorthand)
3 Point severity Severity ratings (0-8)

Annotation Workflow

  1. Open: Sound and TextGrid together (View & Edit)
  2. Transcript tier: Type verbatim transcription
  3. Dysfluency tier: Select region, type shorthand (e.g., R:isyl)
  4. Severity tier: Click at dysfluency midpoint, type severity number
  5. Save: Save → Save as text file

Recommended Display Settings

For optimal dysfluency identification:

Setting Audacity Praat
View mode Waveform + Spectrogram Show spectrogram
Zoom level 50-200 ms visible 0.5-2 s window
Spectrogram range 0-8000 Hz 0-5000 Hz
Time precision milliseconds Show duration

Tip: Spectrogram View

Spectrogram view is especially helpful for:

  • Identifying prolongation boundaries (sustained formant patterns)
  • Detecting audible blocks (low-intensity noise patterns)
  • Distinguishing repetition units (repeated spectral patterns)
