Annotator Guidelines

Complete guide for annotating stuttering and dysfluency using SAML. From quick start to advanced edge cases.

5 5-Minute Overview

What is SAML?

SAML (Stuttering Annotation Markup Language) is an XML-based format for annotating stuttered speech. It extends SSML with elements specifically designed to capture dysfluency events.

What do annotators do?

Annotators listen to audio recordings and mark three types of dysfluency events:

Repetitions

Sounds, syllables, or monosyllabic words repeated

"b-b-baby", "I-I-I"
Prolongations

Sounds stretched beyond normal duration

"sssssnake"
Blocks

Speech stops involuntarily (silent or audible)

"[pause]...The"

Core Workflow

  1. 1 Listen to an audio chunk (typically 5-15 seconds)
  2. 2 Identify any dysfluency events
  3. 3 For each event: determine type, measure duration, assign severity (0-8)
  4. 4 Transcribe using shorthand notation or full XML

Key Principle

Only tag involuntary dysfluencies characteristic of stuttering. Do not tag normal hesitations, filler words (um, uh), or intentional repetitions for emphasis.

Severity Scale (0-8)

All dysfluency events require a severity rating based on duration. Measure from start of dysfluency to end (excluding the target word).

Rating Duration Remember As
0< 0.5sHalf second or less
10.5 - 1sUnder one second
21 - 2sOne to two seconds
32 - 3sTwo to three seconds
43 - 5sThree to five seconds
55 - 10sFive to ten seconds
610 - 30sTen to thirty seconds
730 - 60sHalf minute to one minute
8≥ 60sOne minute or more

Tip: Boundary Values

Boundary values go to the higher rating. Exactly 1.0 seconds = severity 2, not 1.

Repetition Type Decision Guide

?

Is the repeated unit a complete syllable or whole monosyllabic word?

YES → Syllable Repetition (SR)

Examples: "ba-ba-baby", "dog-dog-dog", "I-I-I"

NO → Incomplete Syllable (ISR)

Examples: "b-b-baby", "do-do-dog", "st-st-store"

?

Are BOTH ISR and SR present in the same repetition sequence?

YES → Mixed Repetition

Example: "t-t-to-to-toffee" (incomplete "t-t" + complete "to-to")

Common Mistake

"d-d-do-do-dog" is ISR, not Mixed. Even though lengths vary, all units are incomplete syllables. Mixed requires both incomplete AND complete syllable units.

Block Classification

By Audibility

Inaudible (I)

Complete silence - no sound during block

Audible (A)

Can hear tension, struggle, or partial sounds

By Sound Type

Consonant (C)

Block before consonant-initial word

Vowel (V)

Block before vowel-initial word

IBC

Inaudible Block Consonant

ABC

Audible Block Consonant

IBV

Inaudible Block Vowel

ABV

Audible Block Vowel

Tip: Phonetic Onset

Classify by phonetic onset, not spelling. "Honest" starts with vowel /o/ (silent H), so it would be IBV/ABV.

Best Practices

Do

  • ✓ Listen to each segment multiple times
  • ✓ Use waveform + spectrogram view for precision
  • ✓ Measure duration from waveform, not perception
  • ✓ Use graphemic (written) representation
  • ✓ When uncertain, use lower severity rating
  • ✓ Chunk audio into 5-15 second segments

Don't

  • ✗ Tag filler words (um, uh, like, you know)
  • ✗ Tag intentional repetitions for emphasis
  • ✗ Tag multisyllabic word repetitions
  • ✗ Use phonetic spelling (use "c" not "k" for "car")
  • ✗ Include target word in duration measurement
  • ✗ Break chunks in the middle of a dysfluency
