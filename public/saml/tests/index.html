Comprehensive test cases for validating SAML documents against the XML Schema.

How to Validate SAML Documents

1 Using xmllint (Recommended) 

# Validate a document
xmllint --noout --schema saml.xsd your_file.xml

# Validate with verbose output
xmllint --schema saml.xsd your_file.xml

xmllint comes with libxml2, pre-installed on most systems.

2 Using Python (lxml) 

from lxml import etree

schema = etree.XMLSchema(etree.parse('saml.xsd'))
doc = etree.parse('your_file.xml')
schema.validate(doc)  # Returns True/False

Install with: pip install lxml

3 Using Java (JAXP) 

SchemaFactory factory = SchemaFactory
    .newInstance(XMLConstants.W3C_XML_SCHEMA_NS_URI);
Schema schema = factory.newSchema(new File("saml.xsd"));
Validator validator = schema.newValidator();
validator.validate(new StreamSource(xmlFile));

4 Using SAML Converter 

# Convert and validate in one step
python converter.py input.txt -o output.xml \
    --validate --schema saml.xsd
