Shorthand Converter

CLI tool and Python library for converting SAML shorthand notation to validated XML documents. Supports batch processing, custom metadata, and schema validation.

Quick Start

Convert a single file:

python3 converter.py input.txt -o output.xml

Convert with validation:

python3 converter.py input.txt -o output.xml --validate --schema saml.xsd

Batch convert directory:

python3 converter.py --batch input_dir/ output_dir/

Convert from stdin:

echo "I ( [R:isyl 1] w- w- ) was going." | python3 converter.py -

Python Library Usage

Example: Using as a Python module 
from converter import ShorthandConverter, convert_file

# Simple conversion
converter = ShorthandConverter()
result = converter.convert("I ( [R:isyl 1] w- w- ) was going.")
print(result.xml)

# With metadata
converter = ShorthandConverter(
    file_id="recording-001",
    speaker_id="SPK01",
    task="reading",
    speaker_age="adult",
    speaker_severity="moderate"
)
result = converter.convert(text)

# Check for warnings/errors
if result.warnings:
    print("Warnings:", result.warnings)

# Convert file directly
result = convert_file("input.txt", "output.xml")

Command Line Options

Option Description
-o, --output Output XML file path
--batch INPUT OUTPUT Batch convert directory
--validate Validate output against XSD
--schema PATH Path to SAML XSD file
--file-id ID File identifier for metadata
--speaker-id ID Speaker identifier
--task TYPE Task type (reading, conversation, etc.)
--speaker-age GROUP Age group (child, adolescent, adult)
--speaker-severity LEVEL Severity (mild, moderate, severe)
--stats Show dysfluency statistics
--strict Reject legacy syntax
