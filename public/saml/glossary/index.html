Glossary

Terms and definitions used in SAML and stuttering annotation.

A B C D G I N P R S X

A

ABC (Audible Block Consonant)
A block before a consonant-initial word where audible tension, struggle, or partial sounds can be heard. Compare with IBC.
ABV (Audible Block Vowel)
A block before a vowel-initial word where audible tension, struggle, or partial sounds can be heard. Compare with IBV.

B

Background Sounds
Environmental sounds present in a recording that are not produced by the speaker. Annotated using <saml:background>. Categories include traffic, environmental, human activity, indoor/household, and technical sounds.
Block
A dysfluency where airflow or voicing stops completely. Can be audible (with tension/struggle sounds) or inaudible (silent). Classified by the sound type of the following word (consonant or vowel).

C

Continuant
A speech sound that can be prolonged without change in quality. Includes fricatives (s, f, sh), nasals (m, n), approximants (l, r, w), and vowels. Only continuants can have prolongations.

D

Disfluency
Normal, typical interruptions in speech that occur in all speakers. Includes interjections (um, uh), phrase repetitions, multisyllabic word repetitions, and revisions. NOT tagged in SAML. Compare with Dysfluency.
Dysfluency
Stuttering-specific speech interruptions that are involuntary. Includes incomplete syllable repetitions (ISR), syllable repetitions (SR), mixed repetitions, prolongations, and blocks. Tagged in SAML. Compare with Disfluency.

G

Grapheme
A written letter or letter combination representing a speech sound. SAML uses graphemic representation (written form) rather than phonetic symbols. Example: "c" for "car", not "k".

I

IBC (Inaudible Block Consonant)
A block before a consonant-initial word with complete silence (no audible tension). Compare with ABC.
IBV (Inaudible Block Vowel)
A block before a vowel-initial word with complete silence (no audible tension). Compare with ABV.
Incomplete Syllable Repetition (ISR)
A repetition where the repeated unit does not form a complete syllable. Includes single sound repetitions ("b-b-baby") and partial syllable repetitions ("do-do-dog"). Varying lengths ("d-d-do-do-dog") are still ISR, not mixed.
Interjections
Filler words or sounds inserted during speech (um, uh, like, you know). These are normal disfluencies and should NOT be tagged - write as plain text only.

N

Namespace
An XML mechanism for distinguishing elements from different vocabularies. SAML uses namespace https://shunyalabs.ai/2026/saml with prefix saml:.
Non-speech Sounds
Sounds produced by the speaker that are not linguistic content. Marked using <saml:nonspeech>. Types include laugh, cough, sigh, cry, clear-throat, sniff, inhale, exhale, and other.

P

Prolongation
A dysfluency where a continuant sound is stretched longer than normal. Only possible on sounds that can be sustained (fricatives, nasals, vowels, etc.). Position can be initial, medial, or final.

R

Repetition
A dysfluency where an incomplete syllable, syllable, or monosyllabic word is repeated involuntarily. Classified as ISR (incomplete-syllable), SR (syllable), or mixed.

S

Severity
A rating from 0-8 based on dysfluency duration. 0 = <0.5s, 1 = 0.5-1s, 2 = 1-2s, etc. Boundary values go to higher rating. Required for all dysfluency elements.
SSML
Speech Synthesis Markup Language - a W3C standard for controlling text-to-speech. SAML extends SSML with custom elements for stuttering annotation.
Syllable Repetition (SR)
A repetition where complete syllables or monosyllabic words are repeated. Examples: "ba-ba-baby", "dog-dog-dog", "I-I-I".

X

XSD
XML Schema Definition - a formal specification of XML document structure. SAML provides an XSD file (saml.xsd) for validating SAML documents.
